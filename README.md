# Raspberry Pi IO-LINK Hat

## 概要

Raspberry Pi と IO-Link 基板を接続するためのブリッジ基板です．

24V 電源で動作し，IO-Link 機器と Raspberry Pi への電源の供給を行います．

![Raspberry Pi IO-LINK Hat](img/RPi_IO-Link_Hat.jpg)
